### How to use?
You can use a function by calling it in your code. For example, if you have a function like this:
<code>def test():
    print("hi")
**Output:**
 test()</code>
<code>hi</code>
You can also use them multiple times like this:
<code>test()
test()</code>
**Output:**
<code>hi
hi</code>
